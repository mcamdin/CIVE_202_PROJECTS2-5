# Civil Engineering Coding Projects (CIVE 202)

## Overview  
This repository contains a collection of Python-based projects completed for CIVE 202. Each project focuses on applying civil engineering concepts using code to automate calculations, analyze data, and visualize results.

These projects show how programming can be used alongside engineering principles to improve efficiency and better understand real-world problems.

---

## Projects Included  

### Project #2 – Concrete Mix Design  
This project automates concrete mix design calculations based on NDOT standards. The code calculates material weights and volumes (cement, fly ash, water, and aggregates) for a 1 cubic yard mix. Functions were created to simplify and organize each step of the process.

---

### Project #3 – Transportation Data Analysis  
This project analyzes transportation-related data using Python. Pandas was used to organize and filter data, while Matplotlib was used to create graphs such as boxplots and time-based visualizations. The goal was to identify trends and clearly present results.

---

### Project #4 – Risk Analysis (FEMA National Risk Index)  
This project evaluates natural disaster risk for Montana and Wyoming using FEMA’s National Risk Index dataset. The analysis includes mapping risk scores, comparing regions, and creating a custom risk metric to explore differences in how risk can be defined.

---

### Project #5 – Steel Beam Analysis  
This project uses Python to analyze a steel beam using statics principles. The code calculates reaction forces, shear forces, and bending moments, then determines whether the beam meets basic design requirements.

---

## How to Run  
1. Open any `.ipynb` file in Jupyter Notebook  
2. Run each cell from top to bottom  
3. View outputs and graphs generated in the notebook  

---

## Purpose  
The purpose of these projects is to demonstrate how coding can be applied in civil engineering to:
- Automate repetitive calculations  
- Analyze and interpret data  
- Visualize engineering results  
- Apply concepts from class in a practical way
